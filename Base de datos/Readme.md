
# 🏥 Pharma Reverse - Bases de datos

Este repositorio contiene el diseño e implementación de la base de datos para **Pharma Reverse**, una solución técnica para la gestión de devoluciones y reciclaje de medicamentos.

## 📋 Descripción del Proyecto
El sistema permite conectar Farmacias con Centros Logísticos para procesar medicamentos caducados o retirados del mercado, garantizando la trazabilidad por lote y el cumplimiento de la normativa ambiental.

## 🛠️ Estructura de la Base de Datos
La base de datos se ha desarrollado en MySQL e incluye:
- **Gestión de Usuarios:** Roles y credenciales.
- **Trazabilidad:** Seguimiento desde la farmacia de origen hasta la planta de procesamiento.
- **Integridad:** Uso de claves foráneas y restricciones para evitar datos huérfanos.

## 📂 Contenido del Repositorio
- `pharma_reverse.sql`: Script completo de creación y datos de ejemplo.
- `/pdf`: Documentación del análisis y diagrama E/R.
